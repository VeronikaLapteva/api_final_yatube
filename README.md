# Описание проекта API_FINAL_YATUBE

Проект представляет социальную сеть для публикации личных дневников. 
Это сайт, на котором можно создать свою страницу. Если на нее зайти, то можно посмотреть все записи автора.
Пользователи смогут заходить на чужие страницы, подписываться на авторов и комментировать их записи. 

# Используемые технологии

Python 3.9, Django 3.2.16, Django REST Framework (DRF), REST API, CSRF, Paginator, Simple-JWT, Djoser

## Как запустить проект:
- Клонировать репозиторий и перейти в него в командной строке:
```
https://github.com/VeronikaLapteva/api_final_yatube.git
```
```
cd api_final_yatube
```
- Cоздать и активировать виртуальное окружение:
```
py -m venv venv
```
```
source venv/scripts/activate
```
- Установить зависимости из файла requirements.txt:
```
pip install -r requirements.txt
```
- Выполнить миграции:
```
python manage.py migrate
```
- Запустить проект:
```
python manage.py runserver
```
